    Unity Pause Menu

    1000 lines of well documented c# code.

    Overview

    My open source pause menu was originally made for my game, New Horizons. In order to create this menu I largely implemented the Unity QualitySettings into a custom manager, which allows the gamer to fully customize their experience. In addition, I utilized Sketch to create UI themes and Unity animation system in order to deliver an aesthetically pleasing asset.

    The Audio Panel

    The audio panel was designed with a clean look in mind. Control over the master volume was achieved through modifying the AudioListener Volume. Control over Music and Sound Effects was achieved through modifying the AudioSource volume by iterating over arrays of audio sources and modifying each one by a factor of "f".

    The Video Panel

    The video panel was designed to be robust and easy to use. It was achieved through implementing the QualitySettings. Terrain settings were achieved through modifying the terrain parameters. The aniamtions were achieved through Unity's animation system.

    Project hosting on Github

    From the beginning of this project, I knew I wanted to make this open source. Naturally, I chose Github to host my project. In order to keep this project a competitive option to paid menus, I keep the repo updated with bug fixes and new featrues.
